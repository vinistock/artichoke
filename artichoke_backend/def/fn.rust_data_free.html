A generic implementation of a Free function for mrb_values that store an owned copy of an Rc smart pointer.

This function calls Rc::from_raw on the data pointer and drops the resulting Rc.

Safety

This function assumes that the data pointer is to an Rc<RefCell<T>> created by Rc::into_raw. This fuction bounds T by RustBackedValue which boxes T for the mruby VM like this.

This function assumes it is called by the mruby VM as a free function for an MRB_TT_DATA.
