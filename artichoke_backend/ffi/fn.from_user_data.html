pub unsafe fn from_user_data(
    mrb: *mut mrb_state
) -> Result<Artichoke, InterpreterExtractError>

Extract an Artichoke interpreter from the user data pointer on a sys::mrb_state.

Calling this function will increase the Rc::strong_count on the Artichoke interpreter by one.

Safety

This function assumes that the user data pointer was created with Rc::into_raw and that the pointer is to a non-free'd Rc<RefCell<State>>.
