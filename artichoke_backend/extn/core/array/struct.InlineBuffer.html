pub unsafe fn set_len(&mut self, new_len: usize)[src]

Set the vector's length without dropping or moving out elements

This method is unsafe because it changes the notion of the number of "valid" elements in the vector. Use with care.

Safety

  • new_len must be less than or equal to capacity().
  • The elements at old_len..new_len must be initialized.

Examples

This method is primarily used when mutating an InlineBuffer via a raw pointer passed via FFI into the mruby VM.

See the ARRAY_PTR macro in C.
